    Discover the closer Mlab server available or using geo location to get a random server from a list of the closer servers.

    Declaration

    Swift

    public static func discover<T: URLSessionNDT7>(session: T = URLSession.shared as! T,
                                               withGeoOptions geoOptions: Bool,
                                               retray: UInt = 0,
                                               geoOptionsChangeInRetray: Bool = false,
                                               useNDT7ServerCache: Bool = false,
                                               _ completion: @escaping (_ server: NDT7Server?, _ error: NSError?) -> Void) -> URLSessionTaskNDT7

    Parameters

    session

    URLSession object used to request servers, using URLSession.shared object as default session.
    geoOptions

    true to use a list of servers based in geo location, otherwise the function will work trying to get the closer server.
    retray

    true to use a list of servers based in geo location, otherwise the function will work trying to get the closer server.
    geoOptionsChangeInRetray

    Alternate geo options for retray.
    useNDT7ServerCache

    uses the NDT7 Server cached if exist and if the request/s fail.
    completion

    callback to get the NDT7Server and error message.
    server

    NDT7Server object representing the Mlab server.
    error

    if any error happens, this parameter returns the error.

NDT7APPInfo

public struct NDT7APPInfo : Codable

NDT7APPInfo is an optional object only included in the measurement when an application-level measurement is available.

  • elapsedTime

    elapsed (an Int64) is the time elapsed since the beginning of this test, measured in microseconds.

    Declaration

    Swift

    public let elapsedTime: Int64?
  • numBytes

    numBytes (an Int64) is the number of bytes sent (or received) since the beginning of the specific test. Note that this counter tracks the amount of data sent at application level. It does not account for the protocol overheaded of the WebSockets, TLS, TCP/IP, and link layers.
